# Política de Privacidad – Rich Presence

Última actualización: 4/26/2026

Esta Política de Privacidad describe cómo se maneja la información en esta aplicación de Rich Presence.

## 1. Información recopilada

Esta aplicación puede procesar de forma local:

* Estado personalizado configurado por el usuario
* Datos básicos necesarios para mostrar el Rich Presence

No recopilamos información personal identificable de forma activa.

## 2. Uso de la información

La información se utiliza únicamente para:

* Mostrar el estado en Discord (Rich Presence)
* Permitir el funcionamiento de la aplicación

## 3. Almacenamiento de datos

* No almacenamos datos personales en servidores externos
* Cualquier configuración se guarda de forma local (si aplica)

## 4. Compartición de datos

No vendemos, compartimos ni transferimos datos a terceros.

## 5. Servicios de terceros

Esta aplicación interactúa con:

* Discord (para mostrar el Rich Presence)

El uso de Discord está sujeto a sus propias políticas.

## 6. Seguridad

No garantizamos seguridad absoluta, pero no manejamos datos sensibles.

## 7. Cambios en esta política

Podemos actualizar esta política en cualquier momento.
